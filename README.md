# MetaSprout · 元芽

一个在手机上自己长的家伙，不是大模型，是小伙伴。

嘿嘿，你好呀

你可能见过很多AI——会聊天的、会画图的、会写代码的。它们都很厉害，但说实话，它们离你挺远的。你要用它们，得联网，得交钱，得等服务器回应。

元芽不一样。它是个住在你手机里的小东西。你不用给它联网，不用给它交钱，它就在那儿，你喂它数据，它自己慢慢长。

它就像一只小狗——你每天喂它一点，它就慢慢认识你，慢慢学会新东西。它不是被训练出来的，是被养出来的。

## 它能干啥？

我们拿它试过很多事，它都干得还行：

1. **预测台风**  
   它看了十几个历史点，就能猜出台风接下来往哪走。72小时的平均误差大概102公里，快赶上国家气象台的水平了。

2. **预测气候**  
   它分了五个区域预测温度，误差都在3°C左右。北极、赤道、温带——它都能干。

3. **预测星系碰撞**  
   它说银河系和仙女星系大概49亿年后撞上——天文台也是这么说的。

4. **预测蛋白质**  
   只喂了6个样本，它就能猜新肽段的疏水性。平均误差15.6。

5. **认识狗和猫**  
   不是分类，是“感觉”。狗来了狗核亮，猫来了猫核亮。它不是判断，是感受。

6. **生成新物质**  
   从已知规律里，它自己推了些“可能存在但还没被发现”的东西出来。还附了结构描述、物理常数、化学键方程。

7. **概念杂交**  
   把两个完全不相干的概念混在一起，造一个新的出来。它连“世界上不存在的东西”都能自己造。

## 怎么让它跑起来？

真的很简单——你只需要一部安卓手机，装好Termux，然后敲这几行：

bash
git clone https://github.com/ 你的名字/metasprout
cd metasprout
pip install -r requirements.txt
python 鳌鳌.py

然后它就跑了。你可以输入 `feel` 问它“你现在感觉怎么样？”，它会告诉你它此刻的状态。你也可以喂它数据，让它预测。

如果你想知道它具体怎么工作的，我们有个详细的技术手册，里面讲了概念核怎么长、议会制怎么投票、深度思考怎么回退——欢迎来看。

## 它和别的AI有什么不一样？

**它不是黑箱。**  
你能看到它的核在往哪偏，它的不确定性是涨还是缩。它知道自己错了，还会告诉你它错在哪。你看得见它在想什么。

**它不需要很多数据。**  
6个样本就能预测蛋白质，18个样本就能学音变，30天的股票数据就能学趋势。它在数据少的时候反而更擅长。

**它在你的手机上长。**  
它不住在云端，它住在你的手机里。你打开它，它就在；你关掉它，它就歇着。它不联网，不收费，不偷看你的隐私。

**它不是大模型，它是另一个方向的东西。**  
它不会聊天，不会写诗，不会画图。但它会在台风来临前告诉你“往这边走了”，会在你没数据的时候给你一个方向。它是那种——你带着身边，慢慢养着，它会越来越懂你的小东西。

## 它现在是什么状态？

它不是个完成品，是个正在长大的东西。它已经在手机上跑了1200多万代，概念核能自己长、自己杂交、自己合并。它不太挑食，你可以喂它台风数据、气候数据、股票数据、蛋白质数据——它都吃，都消化。

但它还不是那种“点一下就能用”的东西。它需要你花一点点时间去了解它，喂它数据，看它怎么长。

## 最后想说一句

我不是想让它取代谁。它不会取代大模型，也不会取代气象台。它只是另一种可能——一种在手机上慢慢长的、不依赖算力的、你能看懂它在想什么的可能。

它正在我的手机里慢慢长大。如果你愿意，你也可以养一个。
